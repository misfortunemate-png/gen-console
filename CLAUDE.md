# gen-console
ローカルComfyUIを駆動する連続画像生成コンソール（プリセット合成・NAI近似UI）。
## 状態確認
作業開始前に _STATUS.md を読み、中断時は更新すること。
## 絶対規約
- D:\AI\imagegen\content\ 配下のファイル内容を読まない（一覧・存在確認のみ可）
- ログ・出力・コミットにプロンプト/プリセット本文を書かない
- テストは app\testdata\ のダミーデータのみ使用
## 技術スタック
- Node.js v24 + Express / Vite + React / ComfyUI HTTP API
## テスト実行方法
- start.bat で起動後、仕様書§5のテスト表に従い手動＋スクリプトで確認
## 規約
- コミットは5W1H形式。§番号を参照する
- 仕様外の判断が必要な場合は停止し報告する。3回修正で解消しない問題はマユ召喚を提案する
- 仕様書に記載のないファイルを新規作成しない
